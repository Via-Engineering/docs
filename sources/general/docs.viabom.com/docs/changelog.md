# Changelog

Copy page

Notable product updates, organized by release and date.

[v0.01.25 · April 14, 2026](https://docs.viabom.com/#v00125-april-14-2026)

**Search, part details, and workspace pages now feel smarter and more dependable.**

- Search adapts to your current project context.
- Part pages include built-in datasheet calculators.
- Smaller screens and library links behave more reliably.

[v0.01.24 · April 12, 2026](https://docs.viabom.com/#v00124-april-12-2026)

**Search and part details now do more of the work for you.**

- Smarter search recovery after no-match queries.
- Results better reflect the part or project in view.
- Part pages now include built-in datasheet calculators.

[v0.01.23 · April 5, 2026](https://docs.viabom.com/#v00123-april-5-2026)

**Finding, inspecting, and managing parts now feels faster and more informative across the app.**

- Richer quick look with diagrams and curves.
- Smarter search tags with better matches.
- Faster project flows and cleaner shortcuts.

[v0.01.22 · March 29, 2026](https://docs.viabom.com/#v00122-march-29-2026)

**Release notes are easier to find and read.**

- New in-app changelog page for release notes.
- Find updates from the user menu and footer.
- Cleaner full-page reading for changelog and blog.

[v0.01.21 · March 23, 2026](https://docs.viabom.com/#v00121-march-23-2026)

**Part exploration feels faster and more connected.**

- Part pages and quick look now pull richer component details automatically.
- Series selectors make it easier to compare nearby variants without restarting searches.
- Reliability improvements smooth out search, settings, and data loading flows.

[Shortcuts & Feedback\\ \\ Move faster with keyboard shortcuts and report issues clearly when search or part data is wrong.](https://docs.viabom.com/docs/shortcuts-and-feedback)
